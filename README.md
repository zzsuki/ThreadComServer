# 多线程通信服务器

给予多线程的通信服务端，从配置中读取端口信息，并进行监听；可以接收服务器发送的任意信息，并返回相同的内容

## 使用方法

```python
python3 main.py
```

## 配置文件

```python
PORT_MAP = {
    "Modbus": {
        "TCP": 502,
        "UDP": None,
    },
    "S7COMM": {
        "TCP": 102,
    },
    "FINS": {
        "TCP": 9600,
    },
    "DNP3": {
        "TCP": 20000,
    },
    "OpcUA": {
        "TCP": 48400,
    },
    "IEC104": {
        "TCP": 2404,
    },
    "BACnet": {
        "UDP": 47808,
    },
    "MQTT": {
        "TCP": 1883
    }
}
```

目前协议名称暂时没什么作用，后续会根据协议名称设置对应的handler，对不同协议的请求进行响应，从而模拟不同应用程的通信
